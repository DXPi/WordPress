== Description ==

This plugin generates a multi-effect slider for wordpress based on jQuery Cycle Plugin.

Compatible with all Wordpress versions (since 2.3 to 3.1.2).

== Installation ==

Copy the wp-slider directory int wp-content/plugins and activate it.
